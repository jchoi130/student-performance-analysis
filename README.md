# Student Performance Analysis

## Overview
This project analyses student performance data to identify trends and factors affecting academic results.

The analysis was conducted using Microsoft Excel, focusing on demographic and behavioural variables.

---

## Project Context
This project was completed as part of the **Professional Practice in Data Analytics** course.

The project achieved a **Distinction grade**, demonstrating strong analytical and interpretation skills.

---

## Dataset
The dataset includes:
- Gender  
- Parental level of education  
- Lunch type  
- Test preparation course  
- Math, reading, and writing scores  

---

## Key Analysis

The following factors were analysed:

- Gender vs performance  
- Test preparation vs performance  
- Lunch type vs performance  
- Parental education vs performance  

---

## Key Insights

- Students who completed test preparation generally performed better  
- Students with standard lunch tended to achieve higher scores  
- Parental education level showed a positive correlation with student performance  
- Gender differences in performance were minimal
